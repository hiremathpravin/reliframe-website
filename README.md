# ReliFrame Website

Marketing website for [ReliFrame](https://reliframe.com) — a unified reliability and safety engineering platform.

## Stack
- Plain HTML/CSS/JS — no build step
- Deployed via Cloudflare Pages
- Auto-deploys on push to `main`

## Local dev
Open index.html in a browser — no server needed.

## Products
- ReliFrame FTA (live) — ftastudio.app
- ReliFrame GSN (in development)
- ReliFrame FMEA, RBD, Predict (planned)
